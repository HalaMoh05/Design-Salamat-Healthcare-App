# Design-Salamat-Healthcare-App
Salamat (سلامات) — Jordan National Health Records App A comprehensive mobile healthcare application built for the Jordanian government, enabling citizens to securely access and manage their personal medical records through integration with the national identity system (Sanad).
Overview
Salamat is a bilingual (Arabic/English), RTL/LTR-aware React mobile app designed to give Jordanian citizens a single trusted portal for their health data. It prioritizes accessibility for elderly users, government-level security cues, and a clean medical-grade UI.

Key Highlights
16 screens — from biometric login to lab result charts, prescription history, appointment scheduling, and a share/export system
Dual language — full Arabic (RTL) and English (LTR) switching at runtime, with complete translations across all UI strings
Dark & Light mode — system-aware theming with manual override, applied consistently across every component
Accessible design — large touch targets, adjustable font sizes, high-contrast indicators, and simple navigation patterns optimized for elderly users
Medical data visualization — interactive charts for lab results (trends, reference ranges, history)
Secure sharing — real download and share capabilities with toast feedback and error handling
National identity integration — login and identity verification tied to the Sanad system with biometric authentication support
Government trust indicators — official branding, security badges, and verification states throughout
Tech Stack
React · TypeScript · Tailwind CSS · Recharts · Lucide Icons · i18n (custom) · Dark mode via CSS tokens
